# 회차 작업공간

회차 하나는 하나의 worker 작업 단위다.

```txt
episodes/
  yyyy-mm-dd-slug/
    input/
    planning/
    storyboard/
    prompts/
    drafts/
    review/
    final/
```

## 폴더 역할

- `input/`: 사람이 넣는 원본 아이디어와 브리프
- `planning/`: 줄거리, 등장인물, 장면 목록
- `storyboard/`: 컷 구성, 대사, 화면 지시
- `prompts/`: 이미지 생성 프롬프트
- `drafts/`: 생성 중간 이미지
- `review/`: 캐릭터, 대사, 게시 기준 검수
- `final/`: 최종 이미지, 썸네일, 게시 요청 메타데이터
