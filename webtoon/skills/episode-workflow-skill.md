# 회차 제작 워크플로우

## 1. 입력 정리

`input/brief.md`를 읽고 핵심 상황, 등장인물, 독자에게 남길 감정을 정한다.

## 2. 기획

`planning/story-summary.md`, `planning/cast-list.md`, `planning/scene-list.md`를 작성한다.

## 3. 스토리보드

`storyboard/storyboard.md`에 컷 번호, 장면, 대사, 필요한 이미지 방향을 적는다.

## 4. 이미지 프롬프트

`prompts/panel-prompts.md`에 컷별 이미지 생성 프롬프트와 negative prompt를 작성한다.

## 5. 검수

`review/consistency-check.md`, `review/dialogue-check.md`, `review/publish-checklist.md`를 작성한다.

## 6. 최종 산출

`final/`에 최종 이미지, 썸네일, `episode.json`, `publish.md`를 둔다.

## 게시 원칙

worker는 배포하지 않는다. 사람 검수 후 사이트 데이터에 반영한다.
