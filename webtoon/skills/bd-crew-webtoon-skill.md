# BD크루 웹툰 제작 스킬

## 목적

BD크루 세계관에 맞는 캠핑 웹툰 회차를 생성한다.

## 입력

- 회차 아이디어
- 등장 캐릭터
- 주요 캠핑 상황
- 전달할 캠핑 팁 또는 감정

## 출력

- `planning/story-summary.md`
- `planning/cast-list.md`
- `storyboard/storyboard.md`
- `prompts/panel-prompts.md`
- `review/publish-checklist.md`
- `final/episode.json`

## 규칙

- 캐릭터 기준은 `webtoon/characters/`를 따른다.
- 세계관 기준은 `webtoon/world/`를 따른다.
- 기존 기준 파일은 수정하지 않는다.
- 회차 폴더 밖에 산출물을 만들지 않는다.
