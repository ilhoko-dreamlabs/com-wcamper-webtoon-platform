# WCAMPER Webtoon 메뉴/페이지 개편 설계

## 목적

WCAMPER Webtoon의 첫 화면을 작품 목록 중심에서 AI 웹툰 플랫폼 소개 중심으로 전환한다. 독자용 작품 탐색은 별도 `웹툰` 페이지로 분리하고, 작가 모집과 협업 문의는 각각 목적이 분명한 랜딩 페이지로 구성한다.

## 메뉴 구조

| 메뉴 | 경로 | 목적 |
|---|---|---|
| 홈 | `/` | AI 웹툰 플랫폼 소개, 대표 연재, 참여 흐름 안내 |
| 웹툰 | `/webtoons` | 현재 공개/기획 웹툰 탐색 |
| 작가모집 | `/creators` | 기획력과 AI 활용 기반 작가 모집 |
| 협업문의 | `/partnership` | 브랜드 협업, PPL, 광고, 작가 협의 문의 |
| 마이페이지 | `/mypage` | 일반회원 로그인 후 웹툰 보기, 관심작, 피드백 활동 관리 |
| 작가페이지 | `/creator-studio` | 작가 로그인 후 작품, 회차, 피드백, 협업 문의 관리 |

기존 홈의 `최신웹툰`, `신인작가`, `인기웹툰`, `인기작가` 섹션은 `/webtoons` 내부 섹션으로 이동한다.

## 상단바 설계

상단바는 브랜드, 검색, 로그인 진입을 포함하는 유틸리티 영역과 주요 메뉴 영역을 분리한다.

- 1단 유틸리티 영역
  - 브랜드: `WCAMPER WEBTOON`
  - 검색창: 제목/작가 기준 검색, 제출 시 `/webtoons?q={검색어}`로 이동
  - 일반회원 로그인: 비로그인 상태에서 일반회원 인증 진입
  - 작가 로그인: 비로그인 상태에서 작가 인증 진입
- 2단 주요 메뉴 영역
  - 홈
  - 웹툰
  - 작가모집
  - 협업문의

로그인 상태별 버튼 라벨은 인증 연동 후 다음 기준으로 전환한다.

| 상태 | 일반회원 버튼 | 작가 버튼 |
|---|---|---|
| 비로그인 | 일반회원 로그인 | 작가 로그인 |
| 일반회원 로그인 | 마이페이지 | 작가 로그인 |
| 작가 로그인 | 일반회원 로그인 | 작가페이지 |
| 일반회원+작가 권한 보유 | 마이페이지 | 작가페이지 |

정적 MVP에서는 실제 세션 판별이 없으므로 상단 버튼은 각각 `/mypage`, `/creator-studio` 설계 화면으로 연결한다. 인증 연동 시 `auth.wcamper.com`의 세션/권한 상태를 읽어 라벨과 목적지를 전환한다.

## 홈 `/`

홈은 WCAMPER Webtoon이 무엇을 하는 플랫폼인지 설명하는 소개 페이지다.

- 히어로
  - 핵심 문구: AI로 만드는 캠핑 웹툰 플랫폼
  - 설명: 캠핑, 크루, 가족 여행, 브랜드 이야기를 AI 웹툰으로 제작/연재
  - CTA: `웹툰 보러가기`, `작가로 참여하기`
- AI 웹툰 소개
  - 실제 기록, 사진, 대화, 아이디어를 웹툰 회차로 전환
  - 모바일 스크롤에 맞춘 짧은 연재
  - 작가와 AI 제작도구의 협업 구조
- 현재 공개 연재
  - 대표 작품 일부만 노출
  - 전체 탐색은 `/webtoons`로 이동
- 제작 흐름
  - 기획, 콘티/대사, AI 이미지 제작, 검수, 공개 연재
- 참여 유도
  - 작가 모집과 협업 문의로 이동

## 웹툰 `/webtoons`

웹툰 페이지는 독자가 작품과 회차를 탐색하는 페이지다.

- 페이지 헤더
  - 제목: 웹툰
  - 설명: 공개 중인 AI 웹툰과 기획작 안내
- 최신웹툰
  - 공개 회차 최신순
- 신인작가
  - 작가 카드
- 인기웹툰
  - 작품 카드
- 인기작가
  - 작가 카드

작품, 작가, 회차 상세 경로는 기존 URL을 유지한다.

## 작가모집 `/creators`

작가모집 페이지는 그림 실력보다 웹툰 기획력과 AI 활용 의지가 있는 창작자를 초대한다.

- 히어로
  - 핵심 문구: 웹툰기획력으로 작가되기
  - CTA: 작가 지원 문의
- 모집 대상
  - 캠핑/여행/일상 에피소드를 가진 사람
  - 브랜드나 커뮤니티 이야기를 웹툰화하고 싶은 사람
  - AI 도구로 웹툰 제작을 배우고 싶은 사람
  - 스토리, 기획, 대사에 강한 예비 작가
- 제공 내용
  - AI로 웹툰 만들기 교재
  - 작품 기획 템플릿
  - 캐릭터/세계관 정리 가이드
  - 회차 구성/콘티 가이드
  - 공개 연재 페이지 제공
- 참여 흐름
  - 지원/문의, 기획안 정리, 샘플 회차 제작, 검수 및 피드백, 공개 연재

## 협업문의 `/partnership`

협업문의 페이지는 브랜드와 캠핑 관련 사업자가 웹툰형 광고/협업을 제안하는 B2B 페이지다.

- 히어로
  - 핵심 문구: 웹툰으로 브랜드 이야기를 제안하세요
  - CTA: 협업 제안하기
- 협업 유형
  - PPL/제품 노출
  - 캠핑장/장소 협업
  - 작가 협의형 브랜드 에피소드
  - 광고 캠페인 웹툰 제작
- 제안 시 필요한 정보
  - 브랜드/담당자
  - 제품 또는 장소
  - 희망 노출 방식
  - 일정과 예산 범위
  - 필수 고지/검수 조건
- 운영 원칙
  - 독자 경험을 해치지 않는 자연스러운 노출
  - 광고/협찬 여부 고지
  - 작가와 브랜드 간 사전 협의

## 마이페이지 `/mypage`

마이페이지는 일반회원이 웹툰 감상과 피드백 활동을 관리하는 독자용 대시보드다.

- 히어로
  - 핵심 문구: 마이페이지
  - 설명: 일반회원의 감상, 관심작, 피드백 활동 관리
  - CTA: 웹툰 보러가기, 작가로 참여하기
- 회원 활동 요약
  - 최근 본 웹툰
  - 관심작
  - 내 피드백
  - 완독/조회 기록
- 피드백 활동
  - 회차 의견
  - 작품 평가
  - 작가 응원
  - 피드백 반영/답변 상태
- 향후 데이터 연동
  - 최근 본 회차
  - 관심작 구독 상태
  - 인증 회원 피드백 목록
  - 완독률, 재방문, 활동 이력

마이페이지는 독자가 단순 조회자가 아니라 피드백과 관심작 활동을 남기는 회원으로 전환되도록 설계한다.

## 작가페이지 `/creator-studio`

작가페이지는 작가가 작품 운영과 피드백을 관리하는 창작자용 대시보드다.

- 히어로
  - 핵심 문구: 작가페이지
  - 설명: 작품, 회차, 독자 피드백, 협업 문의 관리
  - CTA: 작가모집 보기, 협업문의 보기
- 작품 관리
  - 작품 현황
  - 회차 제작
  - 독자 피드백
  - 협업 제안
- 작가 작업 흐름
  - 기획 등록
  - 샘플 제작
  - 검수 요청
  - 공개 예약
  - 피드백 반영
- 관리 보드
  - 작가 피드백 보드: 회차별 반응, 완독률, 관심작 전환, 피드백 요약
  - 웹툰 관리 보드: 작품 정보, 공개 일정, 썸네일, AI 사용 고지, 검수 상태
- 향후 데이터 연동
  - 작가별 작품 권한
  - 회차 제출/검수 상태
  - 독자 피드백 집계
  - 광고/PPL 협업 문의 상태

작가페이지는 외부 작가 자유 공개가 아니라 검수와 공개 예약을 거치는 운영형 창작 공간으로 설계한다.

## 구현 범위

- SPA 라우터에 `/webtoons`, `/creators`, `/partnership` 추가
- SPA 라우터에 `/mypage`, `/creator-studio` 추가
- 정적 HTML 생성기에 `/webtoons`, `/creators`, `/partnership`, `/mypage`, `/creator-studio` 경로와 페이지별 메타 추가
- 헤더는 브랜드, 검색창, 일반회원 로그인, 작가 로그인, 4개 주요 메뉴로 구성
- 기존 작품/작가/회차 상세 URL 유지
- 검색은 `/webtoons?q={검색어}`에서 회차, 작품, 작가 기준으로 필터링
- 초기 문의는 `mailto:` CTA로 처리하고, 추후 폼/API로 전환
- 초기 로그인 버튼은 설계 페이지로 연결하고, 추후 통합인증 세션에 따라 `마이페이지`, `작가페이지` 라벨로 전환
