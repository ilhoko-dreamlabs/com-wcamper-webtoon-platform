# WCAMPER Webtoon 메뉴/페이지 개편 설계

## 목적

WCAMPER Webtoon의 첫 화면을 작품 목록 중심에서 AI 웹툰 플랫폼 소개 중심으로 전환한다. 독자용 작품 탐색은 별도 `웹툰` 페이지로 분리하고, 작가 모집과 협업 문의는 각각 목적이 분명한 랜딩 페이지로 구성한다.

## 메뉴 구조

| 메뉴 | 경로 | 목적 |
|---|---|---|
| 홈 | `/` | AI 웹툰 플랫폼 소개, 대표 연재, 참여 흐름 안내 |
| 웹툰 | `/webtoons` | 현재 공개/기획 웹툰 탐색 |
| 작가모집 | `/creators` | 기획력과 AI 활용 기반 작가 모집 |
| 협업문의 | `/partnership` | 브랜드 협업, PPL, 광고, 작가 협의 문의 |

기존 홈의 `최신웹툰`, `신인작가`, `인기웹툰`, `인기작가` 섹션은 `/webtoons` 내부 섹션으로 이동한다.

## 홈 `/`

홈은 WCAMPER Webtoon이 무엇을 하는 플랫폼인지 설명하는 소개 페이지다.

- 히어로
  - 핵심 문구: AI로 만드는 캠핑 웹툰 플랫폼
  - 설명: 캠핑, 크루, 가족 여행, 브랜드 이야기를 AI 웹툰으로 제작/연재
  - CTA: `웹툰 보러가기`, `작가로 참여하기`
- AI 웹툰 소개
  - 실제 기록, 사진, 대화, 아이디어를 웹툰 회차로 전환
  - 모바일 스크롤에 맞춘 짧은 연재
  - 작가와 AI 제작도구의 협업 구조
- 현재 공개 연재
  - 대표 작품 일부만 노출
  - 전체 탐색은 `/webtoons`로 이동
- 제작 흐름
  - 기획, 콘티/대사, AI 이미지 제작, 검수, 공개 연재
- 참여 유도
  - 작가 모집과 협업 문의로 이동

## 웹툰 `/webtoons`

웹툰 페이지는 독자가 작품과 회차를 탐색하는 페이지다.

- 페이지 헤더
  - 제목: 웹툰
  - 설명: 공개 중인 AI 웹툰과 기획작 안내
- 최신웹툰
  - 공개 회차 최신순
- 신인작가
  - 작가 카드
- 인기웹툰
  - 작품 카드
- 인기작가
  - 작가 카드

작품, 작가, 회차 상세 경로는 기존 URL을 유지한다.

## 작가모집 `/creators`

작가모집 페이지는 그림 실력보다 웹툰 기획력과 AI 활용 의지가 있는 창작자를 초대한다.

- 히어로
  - 핵심 문구: 웹툰기획력으로 작가되기
  - CTA: 작가 지원 문의
- 모집 대상
  - 캠핑/여행/일상 에피소드를 가진 사람
  - 브랜드나 커뮤니티 이야기를 웹툰화하고 싶은 사람
  - AI 도구로 웹툰 제작을 배우고 싶은 사람
  - 스토리, 기획, 대사에 강한 예비 작가
- 제공 내용
  - AI로 웹툰 만들기 교재
  - 작품 기획 템플릿
  - 캐릭터/세계관 정리 가이드
  - 회차 구성/콘티 가이드
  - 공개 연재 페이지 제공
- 참여 흐름
  - 지원/문의, 기획안 정리, 샘플 회차 제작, 검수 및 피드백, 공개 연재

## 협업문의 `/partnership`

협업문의 페이지는 브랜드와 캠핑 관련 사업자가 웹툰형 광고/협업을 제안하는 B2B 페이지다.

- 히어로
  - 핵심 문구: 웹툰으로 브랜드 이야기를 제안하세요
  - CTA: 협업 제안하기
- 협업 유형
  - PPL/제품 노출
  - 캠핑장/장소 협업
  - 작가 협의형 브랜드 에피소드
  - 광고 캠페인 웹툰 제작
- 제안 시 필요한 정보
  - 브랜드/담당자
  - 제품 또는 장소
  - 희망 노출 방식
  - 일정과 예산 범위
  - 필수 고지/검수 조건
- 운영 원칙
  - 독자 경험을 해치지 않는 자연스러운 노출
  - 광고/협찬 여부 고지
  - 작가와 브랜드 간 사전 협의

## 구현 범위

- SPA 라우터에 `/webtoons`, `/creators`, `/partnership` 추가
- 정적 HTML 생성기에 3개 경로와 페이지별 메타 추가
- 헤더/푸터 메뉴를 4개 메뉴로 단순화
- 기존 작품/작가/회차 상세 URL 유지
- 초기 문의는 `mailto:` CTA로 처리하고, 추후 폼/API로 전환
